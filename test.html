Git là hệ thống kiểm soát phiên bản phân tán mã nguồn mở ( Open Source Distributed Version Control System) được phát triển bởi Linus Torvalds vào năm 2005. Git sẽ tiến hành theo dõi, ghi nhận và lưu lại các sự thay đổi của các file theo thời gian, từ đó giúp người dùng có thể phục hồi file quay về trạng thái trước đó nếu muốn. Vệc lưu trữ code trong một dự án sẽ dễ dàng hơn, ai cũng thể tiến hành clone code từ một repository về máy tính cá nhân. Vì là clone nên sẽ tạo ra bản sao gần như toàn bộ của repository mà máy chủ đang lưu trữ nên nếu máy chủ gặp sự cố thất thoát dữ liệu thì có thể sử dụng bất kì bản sao trên máy tính cá nhân để khôi phục lại dữ liệu trên máy chủ. Ngoài ra, Git còn có tính năng branch giúp phân chia dự án ra thành các nhánh con khác nhau giúp dễ dàng hơn trong việc làm việc nhóm như mỗi nhánh có thể tiến hành thay đổi code mà không sợ ảnh hưởng tới nhánh chính hay các nhánh khác.
